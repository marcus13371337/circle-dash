# Circle-Dash
The circle-ci dashboard (unoffical) in your your terminal.

## Installation
1. Install the package
```
npm install -g circle-dash
```

2. Configure your user by running
```
circle-dash configure
```

3. Start the dashboard by running
```
circle-dash
```
